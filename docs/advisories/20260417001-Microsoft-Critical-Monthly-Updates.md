# Microsoft Critical Monthly Updates - 20260417001

## Overview

Microsoft have published their Monthly Updates addressing multiple critical and known exploited vulnerabilities.

## What is vulnerable?

### Critical Severity

| Product(s) Affected | Version(s) |CVE                                                               | CVSS | Severity |
| --------|--|--------------------------------------------------------- | ---- | -------- |
|Microsoft Power Apps|Versions prior to 3.26032.10.0 | [CVE-2026-26149](https://nvd.nist.gov/vuln/detail/CVE-2026-26149) | 9.0  | Critical |
|Windows IKE Extension|Versions 10.0.14393.0 to 10.0.14393.9060| [CVE-2026-33824](https://nvd.nist.gov/vuln/detail/CVE-2026-33824) | 9.8  | Critical |

### Known Exploitation

| Product(s) Affected | Version(s) || CVE                                                               | CVSS | Severity |
| -----------------------|- | - |---------------------------------------- | ---- | -------- |
| - Microsoft SharePoint Server Subscription Edition <br/> - Microsoft SharePoint Server 2019 <br/> - Microsoft SharePoint Enterprise Server 2016| - Versions prior to 16.0.19725.20210 <br/> - Versions prior to 16.0.10417.20114 <br/> - Versions prior to Microsoft SharePoint Enterprise Server 2016|[CVE-2026-32201](https://nvd.nist.gov/vuln/detail/CVE-2026-32201) | 6.5  | Medium   |

## What has been observed?

Microsoft has detected exploitation of one or more mentioned Vulnerabilities.
The WASOC has not received any reports of exploitation of this vulnerability on Western Australian Government networks at the time of writing.

## Recommendation

The WASOC recommends administrators apply the solutions as per vendor instructions to all affected devices within expected timeframes (refer [Patch Management](../guidelines/patch-management.md)):

- Microsoft Security Updates: <https://msrc.microsoft.com/update-guide/releaseNote/2026-Apr>
